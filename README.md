# ఆంజనేయముపాస్మహే - ద్వితీయ భాగము

A Telugu translation of the second part (chapters 41–80) of **శ్రీ పరాశర సంహిత** (Parāśara Saṃhitā), also known as *హనుమచ్ఛాస్త్రము* — the scriptural treatise on Hanumad-upāsana.

## Background

The original Sanskrit Saṃhitā was edited and published by **Brahmaśrī Ācārya Annadānam Cidambara Śāstri** in 1997. The first part (chapters 1–40) was translated into Telugu by **Brahmaśrī Ācārya Kollūru Avatāraśarma** in 2010 under the same title *Āñjaneyamupāsmahe*. This repository contains the second part (chapters 41–80), translated by **Kollūru Sītārāma Śarma**.
